📚 How to Use This Tutorial

Work through lessons sequentially. Each includes concepts, working examples, and exercises. Run the interactive examples by opening them in a browser or running the included Node.js servers.

🚀 Quick Start

Start with Lesson 1 to understand RUM concepts, then move to Lesson 2 for your first integration. Lessons 10+ require running backend servers.

💡 Pro Tips

Open browser DevTools to inspect network requests and console output. Check the /boomerang%20tutorial/examples/ folder for runnable code samples.
